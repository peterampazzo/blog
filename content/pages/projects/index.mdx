<Project title="Movi by Mobike (Analytics)" date="May 2019 - August 2022">
  The research focus on leveraging [Ride Movi](https://www.ridemovi.com/) (formely Mobike Italy) data, 
  a free floating bike sharing system, to study urban mobility in Padova (Italy), through descriptive and cluster

<Project title="OpenStreetMap POIs" date="September 2021 - December 2021"> 

This Python package allows to query OpenStreetMap API using OverPy and retrieve all nodes and ways included in a area. 
It is possible to load a list of polygons using GeoPandas (JSON, GeoJSON, Shapefile are supported) and loop through them.

The goal is to associate for every area a list of POIs retrived from OpenStreetMap. 
The query will be performed on the bounding box built around the given polygon.

<h3 className="txt-center">[GitHub](https://github.com/peterampazzo/osm-poi)</h3>
</Project>

<Project title="VirtualDojo" date="March 2020 - June 2020"> 

Due to COVID-19, [CoderDojo Padova](http://www.coderdojopadova.it) decided to conduct 
its activities online during the lock-down in Italy. We teamed up with several clubs all over Italy 
and we reached almost 60 participants. I contributed to develop **VirtualDojo**.

This is open source platform which wraps [Jitsi Meet](https://jitsi.org/) through its
public API and offers a realtime dashboard where it is possible to manage multiple users and rooms.

<h3 className="txt-center">[GitHub](https://github.com/virtualdojo/virtualdojo-rooms)</h3>
</Project>
